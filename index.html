¿Que es BrightSign?

Es un sistema de gestión de contenido, el cual se usa para poder publicar imágenes, videos he incluso archivos web (HTML, CSS, JavaScript) en pantallas y de forma remota.
En esta guía de usuario que hemos diseñado para ti, se recopila toda la información necesaria para poder gestionar el contenido de las taquillas y resolver cualquier tema pendiente. También puedes acudir a Brightsign.biz" que es el sitio web oficial para más información.
